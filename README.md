# Ajay Babu S R - Personal Portfolio Website

A responsive, feature-rich personal portfolio website built for Ajay Babu S R, an analytical Full-Stack Developer with a background in Mathematics. This repository contains the front-end code required to render the portfolio, showcasing technical expertise, featured projects, and academic foundations[cite: 1].

## 🚀 Features

*   **Responsive Design**: Fully responsive layout utilizing CSS Grid and Flexbox, optimized across fluid grid breakpoints for multi-device scalability.
*   **Theme Toggling**: An elegant light/dark theme switching system that remembers user preferences using local storage.
*   **Interactive Navigation**: Includes a sticky navigation bar with contextual shadow styling on scroll and a mobile-friendly hamburger menu.
*   **Dynamic UI Elements**: Features a dynamic scroll-to-top button that seamlessly appears after scrolling past 500 pixels.
*   **Mock Form Submission**: Asynchronous contact routing logic that disables the submit button and mimics API post execution delays before delivering a success alert.

## 🛠️ Technologies Used

*   **Markup & Styling**: HTML5, CSS3 (including custom variables and media queries).
*   **Scripting**: Vanilla JavaScript (ES6+) for DOM manipulation and interactive event handling.
*   **Assets**: Integration of FontAwesome (v6.4.0) for UI icons and Google Fonts (Inter) for clean typography.

## 📂 Project Structure

*   `index.html`: The main structural entry point containing the Hero, About, Skills, Projects, Education, and Contact sections.
*   `css/style.css`: Contains core design tokens, global resets, dark/light theming rules, and responsive CSS media queries.
*   `js/main.js`: Houses the logic for the mobile menu toggling, theme switching, dynamic scroll behaviors, and form submission simulation.
